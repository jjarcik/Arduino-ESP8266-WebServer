# Arduino-ESP8266 WebServer

## You need
- ESP8266 - ESP-01
- Arduino Uno
- 3.3V regulator (I used to AMS1117)


## Wiring ESP8266 + Arduino

| ESP8266 | Arduino | 
| ------- |:-------:|
| RX | 3 |
| TX | 2 |

| ESP8266 | AMS1117 | 
| ------- |:-------:|
| VCC | out |
| CH_PD | out |
| GND | GND |


| AMS1117 | Arduino | 
| ------- |:-------:|
| in | Vin |
| GND | GND |
